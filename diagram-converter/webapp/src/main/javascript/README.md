# Diagram Converter Frontend

This is the React frontend for the Diagram Converter web application. It provides a wizard-like UI for uploading, analyzing, and converting Camunda 7 diagrams to Camunda 8 format.

## Development

```bash
# Install dependencies
npm install

# Start development server with hot reload
npm run dev

# Build for production
npm run build
```

The frontend is bundled into the Spring Boot webapp during the Maven build process.
